# ⌨️ GhostSpeed

> Minimalist typing speed test

![HTML](https://img.shields.io/badge/HTML-CSS-JS-orange?logo=html5) ![License](https://img.shields.io/badge/License-MIT-green)

---

## Cara Pakai

**1. Clone repo**
```bash
git clone https://github.com/MrElixir1945/GhostSpeed.git
cd GhostSpeed
```

**2. Buka langsung di browser**

Di terminal / CMD:
```bash
# Linux / macOS
open index.html

# Windows
start index.html
```

Atau cukup **double-click** file `index.html` di file manager.

> Tidak perlu install apapun. Tidak butuh server. Langsung jalan.

---

## Cara Main

1. Pilih bahasa (**ID** / **EN**) dan durasi (**15 / 30 / 60** detik)
2. Mulai ketik — timer otomatis jalan saat huruf pertama ditekan
3. Hasil WPM dan akurasi tampil setelah waktu habis
4. Tekan **ESC** untuk reset kapan saja

Rekor terbaik tersimpan otomatis di browser (localStorage).
